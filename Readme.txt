Source code and dataset associated with submission to AAAI 2025.


SP.py and Steiner.py: These files include classes related to shortest path tasks and minimum Steiner tree problems, respectively.

StoGraph.py and USCO.py: These files provide classes for abstract query-decision regression problems.

QDR_soup.py (located in the QDRSOUP directory): This file contains the implementation of the QDR_Soup algorithm.

data: This directory contains graphs and samples used in our experiments. Please note that due to the 50MB limitation on Supplementary Material, the generated weighted graphs are not included. However, you can generate them using the provided Gen_weibull.py script.
